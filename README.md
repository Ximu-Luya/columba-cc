# 科伦巴商会/columba-cc

这是一个查看雷索纳斯游戏中各地货物实时价格的应用，帮助各位列车长更好地选择货物运输路线，进行更好的交易(倒货)。目前项目正在开发中，计划支持的功能:

- [x] 查看各地货物实时价格与涨跌
- [x] 查看各地货物的历史价格走势（折线图、K线图）
- [x] 查看各地货物的利润（单位利润）
- [x] 根据货物价格与利润进行排序
- [x] 分析当前最高单次利润运输路线和最高进货书利用方案

## 本地运行/开发测试

```sh
npm install
npm run dev
```

### 应用构建/打包

```sh
npm run build
```
